# Changelog

All notable changes to Guanaco will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-02

### Added

- Initial release
- Chat with local Ollama models
- Streaming responses in real-time
- Markdown rendering with syntax highlighting for code blocks
- Document attachment via drag and drop (PDF, TXT, Markdown)
- Persistent chat history stored locally in SQLite
- Model management dialog with auto-download capability
- System prompt customization per chat
- Settings dialog for Ollama host configuration
- Native GTK4/Libadwaita interface following GNOME HIG
- Spanish translation
